# React Starter
Quick start setup for a React project.

## Setup
1. Clone the repo
```bash
git clone https://github.com/volovar/react-starter.git <your-directory-name>
```
1. Move into the directory
```bash
cd <your-directory-name>
```
1. Install packages
```bash
npm i`
```
1. Reset git
```bash
npm run reset-git
```
